task 1

С экрана пользователем вводиться строка. После нажатия кнопки «Выполнить» изменяет первую букву каждого слова в верхний регистр. Кнопка «Перегрузить» - позволяет очистить поле ввода для дальнейшего ввода информации.


task 2

С экрана пользователем вводиться длинная строка. Если символов больше 9, то только первые 9 символов пишутся и добавляется в конец троеточие. Если символов в строке меньше или равно 9, то текст выводится как он есть. Выводит итог в отдельное текстовое поле после после нажатия кнопки: "Выполнить". Кнопка "Перегрузить" - очищает поле ввода для дальнейшего ввода информации.

task 3

С экрана пользователем вводиться длинная строка не менее 7 слов. Массив выносит из каждого слова первые буквы и выводит итог в отдельное тестовое поле красным цветом после нажатия кнопки "Преобразовать". Копка "Очистить" - очищает поле ввода для дальнейшего ввода информации.

task 4

8 небольших цитат мыслителей по 7-10 слов будут появляться на странице случайным образом после нажатия кнопки: «Выполнить»
